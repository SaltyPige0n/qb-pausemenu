# Pause Menu

A simple pause menu. It was from my server but i have no means of opening one.
## Preview

![Preview](https://i.ibb.co/wFpm6MW9/image.png)

## Features
- Open map and settings
- Set rules, Uses scrollbar to prevent design change
- Disconnect from in the menu
- Shows player count (Can be turned off)
- Discord Button (Opens a browser tab)
- Shows player data

## Requirements
- QBCore

## Installation
1. Drop the resource into your resources folder.
2. Ensure the resource in your server.cfg or if you have dropped in in [qb] then no need
